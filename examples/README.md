# Examples

## eprpairs.cpp

A simple pair of entangled qubits in a shared Bell state

```sh
$ ./eprpairs.out

Circuit Matrix:
| 0.707  0       0.707   0     |
| 0      0.707   0       0.707 |
| 0      0.707   0      -0.707 |
| 0.707  0      -0.707   0     |


System starting state:

| 1 |
| 0 |
| 0 |
| 0 |


Our result is:

| 0.707 |
| 0     |
| 0     |
| 0.707 |


The true result is:

| 0.707 |
| 0     |
| 0     |
| 0.707 |


State after measurement:

| 0 |
| 0 |
| 0 |
| 1 |
```
